Koolack is a microblogging service created in the Spring of 2016 by two Notre Dame students enrolled in CSE40822: Cloud Computing. The purpose of this website is not to serve as an actual production microblogging service; it's purpose is to serve as a test application for evaluating performance in the Amazon Web Services. More specifically, we are trying to quantify the speedup attainable by distributing a Django application using Amazon Elastic Beanstalk.

This is the unscaled version of Koolack. It runs on a single t2.micro EC2 instance. To experience the scaled version of Koolack, please visit scaled.koolack.com. If you are interested in our methodology and finding, please view our presentation and final paper.
